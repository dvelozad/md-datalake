## Quick Reference

### Start Backend
```bash
poetry run uvicorn mddatalake.api.main:app --reload --host 0.0.0.0 --port 8080
```

### Start Frontend
```bash
cd frontend && npm run dev
```

### Test Upload
```bash
curl -X POST http://localhost:8080/api/v1/upload \
  -F "project_name=test" \
  -F "run_name=test" \
  -F "files=@file.dump"
```

### Access UI
http://localhost:3000 → Click "Upload Trajectory"

### API Docs
